Linux para iniciantes

Apresentação.

O sistema operativo Linux tem ganhado cada vez mais utilizadores em todo o mundo. Presente em servidores, smartphones, supercomputadores e inúmeros dispositivos eletrónicos, o Linux tornou-se uma das tecnologias mais importantes da informática moderna. No entanto, apesar da sua enorme utilização, muitas pessoas nunca tiveram contacto com este sistema ou acreditam ser demasiado difícil de usar.

Este eBook foi criado precisamente para mudar essa ideia.

O objetivo deste guia é apresentar o Linux de forma simples, clara e acessível, permitindo que qualquer pessoa comece a utilizá-lo, mesmo sem experiência prévia. Ao longo deste livro serão explicados os conceitos fundamentais, bem como os passos necessários para instalar, configurar e utilizar o sistema no dia a dia.

Este livro foi pensado especialmente para vários tipos de leitores.

Primeiro, para pessoas que nunca utilizaram Linux e que têm curiosidade em conhecer este sistema operativo. Muitos utilizadores ouviram falar de Linux, mas não sabem por onde começar. Este guia procura oferecer um caminho simples e progressivo para iniciar essa aprendizagem.

Também foi escrito para utilizadores de Windows que pretendem mudar para Linux. A transição entre sistemas operativos pode parecer complicada, sobretudo quando estamos habituados a um determinado ambiente de trabalho. Por essa razão, o conteúdo deste livro explica as diferenças entre os sistemas e mostra como realizar tarefas comuns de forma prática.

Outro grupo importante são os estudantes de informática. Conhecer Linux tornou-se quase essencial em muitas áreas tecnológicas, incluindo administração de sistemas, programação, redes e cibersegurança. Este eBook fornece uma base sólida que pode servir como ponto de partida para estudos mais avançados.

Finalmente, este livro é também útil para pessoas que querem recuperar computadores antigos. Muitas máquinas que já não conseguem executar versões recentes de outros sistemas operativos podem voltar a funcionar perfeitamente com Linux. Utilizar uma distribuição adequada pode prolongar a vida útil de um computador durante vários anos.

Ao longo deste guia parte-se do princípio de que o leitor começa praticamente do zero. Por isso, não é necessário ter conhecimentos prévios sobre Linux. Assume-se também que o leitor:

  • não sabe o que é uma distribuição Linux
  • nunca utilizou o terminal
  • pode ter receio de cometer erros ou de danificar o computador

Alguns exemplos conhecidos de sistemas operativos são:

  • macOS
  • Linux
  • Android
  • iOS

Ao ligar um computador, o sistema operativo é carregado para a memória e controla todo o funcionamento da máquina.

1.2 O que é Linux.

O termo Linux refere-se ao núcleo de um sistema operativo, chamado kernel. O kernel é o componente central que faz a comunicação entre o ‘software’ e o ‘hardware’ do computador.

Ele controla, por exemplo:

  • o processador
  • a memória
  • os discos
  • os dispositivos de entrada e saída

Em termos simples, o Linux é o “coração" do sistema operativo. No entanto, quando as pessoas dizem que usam Linux, referem-se normalmente a um sistema completo baseado nesse núcleo.

Esse sistema inclui:

  • o kernel Linux
  • ferramentas do sistema
  • programas básicos
  • um ambiente gráfico

A combinação de estes elementos forma aquilo que se chama uma distribuição Linux.

1.3 História do Linux

O Linux surgiu em 1991, criado por um estudante finlandês chamado Linus Torvalds.
Na altura, Linus Torvalds estudava na Universidade de Helsínquia e queria um sistema operativo semelhante ao UNIX para o seu computador pessoal.
Como não estava satisfeito com as alternativas disponíveis, decidiu criar o seu próprio kernel. Inicialmente tratava-se apenas de um projeto pessoal, mas chamou rapidamente a atenção de programadores de todo o mundo.
Linus publicou o código na ‘internet’ e convidou outros programadores a colaborar. Este modelo de desenvolvimento aberto permitiu que o sistema evoluísse rapidamente.
Com o tempo, milhares de programadores passaram a contribuir para o projeto, tornando o Linux num dos sistemas mais estáveis e poderosos existentes.

Capítulo 1: O que é Linux.

1.1 O que é um sistema operativo.

Um sistema operativo é o ‘software’ principal que permite que um computador funcione. É ele que faz a ligação entre o utilizador, os programas e o ‘hardware’ do computador. Sem um sistema operativo, um computador não consegue executar programas nem realizar tarefas úteis.

Entre as principais funções de um sistema operativo estão:

  • gerir a memória do computador
  • controlar o processador
  • permitir o uso de discos e dispositivos de armazenamento
  • gerir periféricos, como teclado, rato, impressoras e placas de rede
  • executar programas e aplicações

Alguns exemplos conhecidos de sistemas operativos são:

  • macOS
  • Linux
  • Android
  • iOS
