linux com Ubuntu


Introdução

O sistema operativo Linux tem ganhado cada vez mais utilizadores em todo o mundo. Presente em servidores, ‘smartphones’, supercomputadores e inúmeros dispositivos eletrónicos, o Linux tornou-se uma das tecnologias mais importantes da informática moderna. No entanto, apesar da sua enorme utilização, muitas pessoas nunca tiveram contacto com este sistema ou acreditam ser demasiado difícil de usar.

Este eBook foi criado precisamente para mudar essa ideia.

O objetivo deste guia é apresentar o Linux de forma simples, clara e acessível, permitindo que qualquer pessoa comece a utilizá-lo, mesmo sem experiência prévia. Ao longo deste livro serão explicados os conceitos fundamentais, bem como os passos necessários para instalar, configurar e utilizar o sistema no dia a dia.

Este livro foi pensado especialmente para vários tipos de leitores.

Primeiro, para pessoas que nunca utilizaram Linux e que têm curiosidade em conhecer este sistema operativo. Muitos utilizadores ouviram falar de Linux, mas não sabem por onde começar. Este guia procura oferecer um caminho simples e progressivo para iniciar essa aprendizagem.

Também foi escrito para utilizadores de Windows que pretendem mudar para Linux. A transição entre sistemas operativos pode parecer complicada, sobretudo quando estamos habituados a um determinado ambiente de trabalho. Por essa razão, o conteúdo deste livro explica as diferenças entre os sistemas e evidência como realizar tarefas comuns de forma prática.

Outro grupo importante são os estudantes de informática. Conhecer Linux tornou-se quase essencial em muitas áreas tecnológicas, incluindo administração de sistemas, programação, redes e cibersegurança. Este eBook fornece uma base sólida que pode servir como ponto de partida para estudos mais avançados.

Finalmente, este livro é também útil para pessoas que querem recuperar computadores antigos. Muitas máquinas que já não conseguem executar versões recentes de outros sistemas operativos podem voltar a funcionar perfeitamente com Linux. Utilizar uma distribuição adequada pode prolongar a vida útil de um computador durante vários anos.

Ao longo deste guia parte-se do princípio de que o leitor começa praticamente do zero. Por isso, não é necessário ter conhecimentos prévios sobre Linux.

Assume-se também que o leitor:
• não sabe o que é uma distribuição Linux.
• nunca utilizou o terminal.
• pode ter receio de cometer erros ou de danificar o computador

Todas as explicações foram escritas com isso em mente. Os conceitos são apresentados gradualmente e cada passo é descrito com clareza, de modo a reduzir ao máximo qualquer dificuldade inicial.

A distribuição Linux utilizada neste livro será o Ubuntu. Esta escolha não é por acaso. Ubuntu é uma das distribuições Linux mais populares do mundo e é amplamente reconhecida pela sua facilidade de utilização. Possui uma grande comunidade de utilizadores, extensa documentação e uma enorme quantidade de software disponível.

Utilizar Ubuntu permite que novos utilizadores aprendam Linux num ambiente estável, moderno e relativamente simples de compreender. Este eBook pretende, acima de tudo, ser um guia prático.

No final da leitura, o objetivo é que o leitor seja capaz de:
• compreender os conceitos básicos do Linux.
• instalar Ubuntu no seu computador.
• utilizar o sistema no dia a dia,
• instalar programas.
• realizar tarefas comuns com confiança.

Começar a usar Linux pode parecer um desafio no início, mas com orientação adequada torna-se um processo acessível e até bastante interessante. Espera-se que este livro seja o primeiro passo para descobrir um sistema operativo poderoso, flexível e cada vez mais relevante no mundo da tecnologia.

Parte I - Compreender o Linux

Capítulo 1 - O que é Linux

1.1 O que é um sistema operativo

Um sistema operativo é o ‘software’ principal que permite que um computador funcione. É ele que faz a ligação entre o utilizador, os programas e o ‘hardware’ do computador.
Sem um sistema operativo, um computador não consegue executar programas nem realizar tarefas úteis.

Entre as principais funções de um sistema operativo estão:

• gerir a memória do computador
• controlar o processador.
• permitir o uso de discos e dispositivos de armazenamento.
• gerir periféricos, como teclado, rato, impressoras e placas de rede.
• executar programas e aplicações.

Alguns exemplos conhecidos de sistemas operativos são:

• Microsoft Windows.
• macOS.
• Linux.
• Android.
• iOS.

Ao ligar um computador, o sistema operativo é carregado para a memória e controla todo o funcionamento da máquina.

1.2 O que é Linux

O termo Linux refere-se ao núcleo de um sistema operativo, chamado kernel .
O kernel é o componente central que faz a comunicação entre o ‘software’ e o ‘hardware’ do computador. Ele controla, por exemplo:

• o processador.
• a memória.
• os discos.
• os dispositivos de entrada e saída.

Em termos simples, o Linux é o “coração" do sistema operativo. No entanto, quando as pessoas dizem que usam Linux, referem-se normalmente a um sistema completo baseado nesse núcleo.

Esse sistema inclui:

• o kernel Linux.
• ferramentas do sistema.
• programas básicos.
• um ambiente gráfico.

A combinação de estes elementos forma aquilo que se chama uma distribuição Linux .

1.3 História do Linux

O Linux surgiu em 1991, criado por um estudante finlandês chamado Linus Torvalds.
Na altura, Linus Torvalds estudava na Universidade de Helsínquia e queria um sistema operativo semelhante ao UNIX para o seu computador pessoal.
Como não estava satisfeito com as alternativas disponíveis, decidiu criar o seu próprio kernel. Inicialmente tratava-se apenas de um projeto pessoal, mas chamou rapidamente a atenção de programadores de todo o mundo.
Linus publicou o código na ‘internet’ e convidou outros programadores a colaborar. Este modelo de desenvolvimento aberto permitiu que o sistema evoluísse rapidamente.
Com o tempo, milhares de programadores passaram a contribuir para o projeto, tornando o Linux num dos sistemas mais estáveis e poderosos existentes.

Atualmente o Linux é usado em:

• servidores da ‘internet’.
• supercomputadores.
• ‘smartphones’.
• routers.
• televisões inteligentes.
• sistemas embarcados.
• computadores pessoais.

O sistema operativo Android , por exemplo, utiliza o kernel Linux. Hoje em dia, o desenvolvimento do Linux continua a ser coordenado por Linus Torvalds com a ajuda de uma grande comunidade internacional.

1.4 ‘Software’ livre e código aberto

O Linux está associado ao conceito de ‘software’ livre e código aberto .

‘Software’ livre significa que os utilizadores têm liberdade para:

• usar o programa para qualquer finalidade.
• estudar como o programa funciona.
• modificar o código.
• distribuir cópias.

O termo código aberto significa que o código do programa está disponível para ser consultado e alterado.

Isto traz várias vantagens:

• maior transparência.
• desenvolvimento colaborativo.
• correção rápida de erros.
• maior segurança.

Grande parte do ‘software’ utilizado em sistemas Linux segue esta filosofia.
É importante notar que ‘software’ livre não significa necessariamente ‘software’ gratuito , embora muitas vezes seja.

1.5 O que é uma distribuição Linux

O kernel Linux por si só não é suficiente para criar um sistema operativo completo.

Para um utilizador poder trabalhar num computador, são necessários vários componentes adicionais, como:

• ferramentas de sistema.
• ‘interface’ gráfica.
• programas básicos.
• gestor de pacotes.
• instalador do sistema.

Uma distribuição Linux reúne estes elementos num sistema pronto a usar. Existem muitas distribuições diferentes, cada uma com objetivos e características próprias.

Algumas das mais conhecidas são:

• Ubuntu.
• Linux Mint.
• Debian.
• Fedora.
• Arch Linux.

Apesar das diferenças, estas distribuições utilizam o mesmo núcleo Linux. No caso deste livro, a distribuição utilizada será o Ubuntu , uma das mais populares e recomendadas para iniciantes.

1.6 Onde o Linux é utilizado

Muitas pessoas pensam que o Linux é usado apenas por programadores ou especialistas, mas na realidade ele está presente em inúmeros dispositivos.

Alguns exemplos incluem:

Servidores da ‘internet’
Grande parte dos sítios ‘web’ e serviços ‘online’ funciona em servidor Linux.
Supercomputadores
A maioria dos supercomputadores mais poderosos do mundo utiliza Linux.
‘Smartphones’
O sistema operativo Android, presente em milhões de ‘smartphones’, baseia-se no kernel Linux.
Equipamentos de rede
Muitos routers domésticos e empresariais utilizam Linux internamente.
Dispositivos inteligentes
Televisões, sistemas automóveis e equipamentos industriais utilizam frequentemente versões adaptadas do Linux.
Computadores pessoais
Cada vez mais utilizadores escolhem distribuições Linux para o uso diário.

1.7 O papel da comunidade

Uma das principais forças do Linux é a sua comunidade.
Milhares de programadores, empresas e utilizadores contribuem diariamente para melhorar o sistema.

Entre as formas de contribuição estão:

• desenvolvimento de ‘software’.
• correção de erros.
• tradução de programas.
• criação de documentação.
• ajuda em fóruns e comunidades.

Esta colaboração global permite que o Linux evolua continuamente e permaneça atualizado.

1.8 Resumo do capítulo

Neste capítulo foram apresentados os conceitos fundamentais necessários para compreender o Linux.

Pontos principais:

• Um sistema operativo é o ‘software’ que controla o funcionamento do computador.
• O Linux é o kernel que serve de base a muitos sistemas operativos.
• Foi criado em 1991 por Linus Torvalds.
• Está associado ao movimento de ‘software’ livre e código aberto.
• As distribuições Linux combinam o kernel com ferramentas e programas para criar um sistema completo.
• O Linux é utilizado em servidores, ‘smartphones’, supercomputadores e computadores pessoais.

Capítulo 2: Escolher a distribuição

2.1 O que é uma distribuição Linux

Como foi explicado no capítulo anterior, Linux é apenas o kernel, ou seja, o núcleo do sistema operativo. Para um utilizador poder realmente trabalhar num computador, são necessários muitos outros componentes.

• ‘interface’ gráfica.
• ferramentas do sistema.
• programas básicos.
• gestor de pacotes.
• instalador do sistema.
• aplicações essenciais.

Uma distribuição Linux é um sistema completo que reúne estes elementos num conjunto pronto a instalar e utilizar.

Cada distribuição inclui:

• o kernel Linux.
• ‘software’ de base.
• um ambiente de trabalho.
• um sistema de instalação.
• um conjunto de programas.

Embora todas utilizem o mesmo núcleo, cada distribuição pode ter diferenças importantes, como:

• aparência do ambiente gráfico.
• programas incluídos por defeito.
• sistema de gestão de ‘software’.
• público-alvo.

Algumas distribuições são feitas para iniciantes, outras para programadores ou administradores de sistemas.

2.2 Porque existem tantas distribuições

Uma das características do Linux é a sua natureza aberta. Como o código é público, qualquer pessoa ou organização pode criar a sua própria distribuição. Isto levou ao aparecimento de centenas de distribuições diferentes, cada uma com objetivos específicos.

Alguns exemplos de objetivos comuns:

Facilidade de utilização
Distribuições pensadas para utilizadores que começam.
Estabilidade
Distribuições usadas em servidores e ambientes profissionais.
Desempenho
Sistemas otimizados para computadores antigos ou com poucos recursos.
Controlo total br Distribuições que permitem ao utilizador configurar todos os aspetos do sistema. Apesar desta diversidade, muitas distribuições partilham uma base comum. Por exemplo, várias distribuições são baseadas noutras já existentes.

2.3 Distribuições Linux populares

Existem muitas distribuições Linux, mas algumas tornaram-se particularmente populares devido à sua qualidade e facilidade de utilização.

Ubuntu

Ubuntu é uma das distribuições Linux mais conhecidas e utilizadas no mundo.
Características principais:

• ‘interface’ moderna e fácil de usar.
• grande comunidade de utilizadores.
• atualizações regulares.
• vasta quantidade de ‘software’ disponível.

É frequentemente recomendada para quem começa.

Linux Mint

Linux Mint é uma distribuição baseada em Ubuntu, conhecida pela sua simplicidade.
Características principais:

• ‘interface’ semelhante ao Windows.
• muito estável.
• fácil para utilizadores iniciantes.

É uma das distribuições favoritas de quem vem do Windows.

Debian

Debian é uma das distribuições mais antigas e respeitadas no mundo Linux.
Características principais:

• grande estabilidade.
• enorme repositório de ‘software’.
• base para muitas outras distribuições.

O próprio Ubuntu é baseado em Debian.

Fedora

Fedora é uma distribuição moderna que inclui tecnologias recentes.
Características principais:

• ‘software’ atualizado.
• forte ligação à comunidade open source.
• sistema estável e seguro,

É frequentemente utilizada por programadores.

Arch Linux

Arch Linux é uma distribuição avançada destinada a utilizadores experientes.
Características principais:

• instalação manual.
• grande controlo sobre o sistema.
• documentação excelente.

Não é recomendada para quem começa.

2.4 Como escolher uma distribuição

Escolher uma distribuição pode parecer difícil no início, mas na realidade a decisão pode ser simplificada considerando alguns fatores.

Facilidade de utilização.
Para iniciantes, é importante escolher uma distribuição com:

• ‘interface’ simples.
• instalação fácil.
• boa documentação,

Comunidade ativa
Uma comunidade grande significa:

• mais tutoriais.
• mais fóruns de ajuda.
• mais soluções para problemas.

Disponibilidade de ‘software’
Algumas distribuições têm mais programas disponíveis ou repositórios maiores.

Estabilidade
Para uso diário, é preferível um sistema estável que funcione bem durante muito tempo sem problemas.

2.5 Distribuições recomendadas para iniciantes

Entre as muitas distribuições existentes, algumas são particularmente adequadas para quem nunca usou Linux.

As mais recomendadas são:

• Ubuntu.
• Linux Mint.
• Zorin OS.
• Ubuntu MATE.

Estas distribuições foram concebidas com foco na simplicidade e na facilidade de utilização.

2.6 Porque este livro utiliza Ubuntu

Neste livro será utilizada a distribuição Ubuntu.
Esta escolha deve-se a várias razões importantes:
Grande comunidade
Ubuntu possui uma das principais comunidades de utilizador Linux do mundo.
Abundância de documentação
A maioria dos tutoriais e guias disponíveis na ‘internet’ utiliza Ubuntu como exemplo.
Facilidade de utilização
O sistema é simples de instalar e usar, mesmo para quem nunca utilizou Linux.
Abundância de ‘software’
Existe uma enorme biblioteca de programas disponíveis.
Base para outras distribuições
Muitas distribuições populares são baseadas em Ubuntu, o que significa que aprender Ubuntu facilita aprender outras distribuições.

2.7 Versões do Ubuntu

Ubuntu possui diferentes versões ao longo do tempo.
Existem dois tipos principais:

Versões LTS (Long Term Support.) Estas versões recebem suporte durante vários anos sendo ideais para:
• uso diário.
• computadores pessoais.
• ambientes de trabalho.

São as versões mais recomendadas.

Versões intermédias
Entre as versões LTS existem versões com suporte mais curto.
Estas versões incluem ‘software’ mais recente, mas não são necessárias para a maioria dos utilizadores.

Pontos principais:

• Existem centenas de distribuições diferentes.
• Cada distribuição tem objetivos e características próprias.
• Algumas distribuições são mais adequadas para iniciantes.
• Ubuntu é uma das distribuições mais populares e fáceis de usar.
