# linux com Ubuntu

**linux com Ubuntu** é um guia gratuito criado para ajudar qualquer pessoa a começar a utilizar Linux de forma simples e prática.

Este projeto foi pensado especialmente para iniciantes que nunca utilizaram Linux ou que pretendem mudar de outros sistemas operativos. O objetivo é explicar os conceitos fundamentais e mostrar, passo a passo, como instalar, configurar e utilizar o sistema no dia a dia.

O guia utiliza a distribuição Ubuntu, uma das mais populares e fáceis de utilizar no mundo Linux. Ao longo do conteúdo são apresentados exemplos claros, explicações simples e procedimentos detalhados que permitem aprender de forma progressiva e segura.

## Público-alvo

Este projeto destina-se principalmente a:

* pessoas que nunca utilizaram Linux
* utilizadores de Windows que pretendem mudar para Linux
* estudantes de informática
* pessoas que querem recuperar computadores antigos

O conteúdo assume que o leitor está a começar praticamente do zero. Não é necessário ter conhecimentos prévios sobre Linux, terminal ou administração de sistemas.

## Objetivo do projeto

O objetivo principal é criar um recurso educativo aberto e acessível que permita a qualquer pessoa:

* compreender o que é Linux
* instalar Ubuntu num computador
* aprender a utilizar o sistema no dia a dia
* instalar e gerir programas
* resolver problemas básicos
